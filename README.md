# yazilimMuhGuncelKonularProject
1. Proje Başlığı

LiverAR: Karaciğer Nakli Sonrası Takip İçin Artırılmış Gerçeklik Uygulaması

2. Proje Özeti

Bu proje, karaciğer nakli geçirmiş hastaların ameliyat sonrası durumlarını daha iyi anlamalarını sağlamak amacıyla geliştirilmiş bir artırılmış gerçeklik (AR) uygulamasıdır. Kullanıcılar, karaciğerin mevcut durumunu 3D model üzerinden inceleyebilir ve sağlık verilerini görsel olarak takip edebilir.

3. Problem Tanımı

Karaciğer nakli sonrası hastalar, vücutlarında gerçekleşen değişimleri anlamakta zorlanmakta ve tedavi sürecini yeterince takip edememektedir. Mevcut sistemler genellikle metin veya 2D verilerle sınırlıdır ve bu durum kullanıcı deneyimini kısıtlamaktadır.

4. Amaçlar
Karaciğerin 3D modelini AR ile görselleştirmek
Hastaların sağlık durumunu daha anlaşılır hale getirmek
Doktor-hasta iletişimini güçlendirmek
Tıbbi verileri görsel ve etkileşimli şekilde sunmak
Eğitim amaçlı kullanılabilir bir sistem geliştirmek
5. Temel Kavramlar
Artırılmış Gerçeklik (AR)
3D Modelleme
Sağlık Bilişimi
Karaciğer Nakli
Hasta Takip Sistemleri
6. Kullanılan Teknolojiler
Unity / Unreal Engine (AR geliştirme)
ARCore / ARKit
Flutter veya React Native (mobil uygulama)
Firebase (kullanıcı yönetimi ve veri saklama)
Node.js / Python (FastAPI) (backend)
3D Modelleme Araçları (Blender vb.)
7. Proje Kapsamı

Bu proje, karaciğer nakli sonrası hastaların sağlık durumlarını AR teknolojisi ile görselleştirmeyi kapsamaktadır. Uygulama, temel hasta verileri ve karaciğer modeli üzerinde odaklanır. Klinik teşhis koyma amacı taşımaz.

8. Beklenen Çıktılar
Çalışan bir mobil AR uygulaması
3D karaciğer modeli
Hasta verilerinin görsel analizi
Kullanıcı dostu arayüz
Prototip demo sistemi
9. Katkıda Bulunanlar

10. Kaynaklar
