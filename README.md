# yazilimMuhGuncelKonularProject
Proje Başlığı
LiverAR: Karaciğer Nakli Sonrası AR Takip Uygulaması
Proje Özeti
Karaciğer nakli sonrası hastaların durumunu AR ile görselleştiren mobil uygulama.
Problem Tanımı
Hastalar ameliyat sonrası süreci anlamakta zorlanmaktadır.
Amaçlar
3D modelleme, hasta takibi, görselleştirme ve iletişim geliştirme.
Temel Kavramlar
AR, 3D Modelleme, Sağlık Bilişimi, Hasta Takip
Kullanılan Teknolojiler
Unity, ARCore/ARKit, Flutter/React Native, Firebase, Node.js
Proje Kapsamı
AR ile görselleştirme, teşhis amaçlı değildir.
Beklenen Çıktılar
Mobil uygulama, 3D model, prototip sistem
Katkıda Bulunanlar
Samet Yasıt
Kaynaklar
Tıbbi makaleler ve teknik dokümanlar
Anahtar Kelimeler
Karaciğer, AR, Sağlık, 3D Model
